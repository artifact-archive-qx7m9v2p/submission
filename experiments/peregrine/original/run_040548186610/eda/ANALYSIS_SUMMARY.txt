KEY FINDINGS
============

1. TREND: Strong non-linear (accelerating) growth
   - Pearson r = 0.941 (p < 0.001)
   - Quadratic R² = 0.961 (best fit)
   - Growth rate increases 6× from early to late period

2. OVERDISPERSION: Extreme (Variance-to-mean ratio = 68.0)
   - Far exceeds Poisson assumption (expected = 1.0)
   - Varies across time: Q1=1.6×, Q2=2.4×, Q3=13.5×, Q4=2.3×
   - MUST use negative binomial or similar

3. AUTOCORRELATION: Very high temporal dependence
   - Lag-1 correlation = 0.989 (p < 0.001)
   - Durbin-Watson = 0.195 (expect ~2 for independence)
   - May need AR structure in model

   - No missing values
   - No problematic outliers
   - Clean integer counts (19 to 272)
   - Evenly spaced time points

RECOMMENDED MODEL
=================

Primary: NEGATIVE BINOMIAL GLM WITH QUADRATIC TREND

Model specification:
  C_i ~ NegBinomial(μ_i, φ)
  log(μ_i) = β₀ + β₁·year_i + β₂·year_i²

Rationale:
  ✓ Handles count data nature
  ✓ Accounts for overdispersion (φ parameter)
  ✓ Captures non-linear accelerating trend (year²)
  ✓ Best AIC (231.8 vs 273.2 for linear)
  ✓ Can be extended with AR(1) if needed

Priors (suggested):
  β₀ ~ Normal(log(100), 1)    # Based on observed mean
  β₁ ~ Normal(0, 1)            # Positive trend expected
  β₂ ~ Normal(0, 0.5)          # Acceleration term
  φ ~ Half-Normal(0, 5)        # Dispersion parameter

Alternative Models (for comparison):
  - Exponential trend: log(μ) = β₀ + β₁·year (R² = 0.929)
  - With AR(1) errors if residual autocorrelation remains
  - Log-normal as continuous approximation

VALIDATION STRATEGY
===================

Model Checking:
  1. Posterior predictive checks (visual + quantitative)
  2. LOO cross-validation (WAIC/LOO-IC)
  3. Residual autocorrelation (check if AR needed)
  4. Dispersion adequacy (Var/Mean in posterior predictive)
  5. Forecast accuracy (hold out last 20% as test set)

Red Flags to Watch:
  - Residuals still show U-shape → Need more flexibility
  - Residual ACF still high → Add AR structure
  - Overdispersion not captured → Check φ parameter
  - Poor calibration → Prediction intervals too narrow/wide

Success Criteria:
  ✓ Residuals appear random (no patterns)
  ✓ Posterior predictive includes observed data
  ✓ LOO-IC better than alternatives
  ✓ Reasonable forecast accuracy
