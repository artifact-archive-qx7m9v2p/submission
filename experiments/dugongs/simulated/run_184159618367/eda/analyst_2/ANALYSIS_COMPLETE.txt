================================================================================
MODEL RECOMMENDATIONS
================================================================================

RANKED BY PREFERENCE:

1. GAUSSIAN PROCESS REGRESSION
   - Best for smooth non-linear relationships
   - Handles heteroscedasticity naturally
   - Provides uncertainty quantification

2. NON-LINEAR LEAST SQUARES (Asymptotic)
   - Formula: Y = a - b*exp(-c*x)
   - Fitted: Y = 2.565 - 1.019*exp(-0.204*x)
   - Best interpretability + performance balance
   - Clear asymptote at Y ≈ 2.57

3. POLYNOMIAL REGRESSION (Quadratic)
   - Good fit (R² = 0.86)
   - Simple to implement
   - Standard inference available

4. LOG-LOG TRANSFORMED LINEAR
   - log(Y) = α + β*log(x)
   - Achieves high linearity (r = 0.92)
   - Simplest approach with good results

AVOID:
  × Simple linear regression (insufficient)
  × High-degree polynomials (overfitting risk)
